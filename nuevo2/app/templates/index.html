PROBANDO DJANGO (RECORDANDO)

    REPASO DE:
  1. creación de proyecto, previo tener instalado django (pip install django), con python manage.py startproject
  2. creación de APPs del proyecto, con python manage.py startapp
  3. inclusión de la APP creada, en el archivo config de la raiz
  4. creación de views.py para incluir vistas (from django.shortcuts import render).
    Se crea una función (def) para cada vista y se incluye el parámetro "request".
    Se retorna render(request,'template.html', parametros en formato diccionario)
  5. creación de carpeta "templates" dentro de la app
  6. para uso de plantilla base se debe inlcuir .. extends 'base.html'.. entre { y % y completar entre los bloques ..block nombre..endblock nombre.. entre { y % (abrir y cerrar)
  7. para "incrustar" un código html desde otro archivo, se agrega ..include 'components/nav.html'.. entre { y %
  8. creación de la carpeta "static" e inclusión dentro de la app
  9. inclusión de archivos css y js en el template base. href="..static 'nombre de archivo'.. entre { y %. Se debe incluir ..load static.. entre { y % al comienzo del documento'
  10. para generar la base de datos se debe ejecutar 'manage.py make migrations' + 'manage.py migrate'
